报错记录

42000错误 42S22错误 
	SQLSTATE[42000]: Syntax error or access violation: 1055 'kf_local.g.group_name' isn't in GROUP BY
原因：在mysql的配置中设置了sql_mode包含ONLY_FULL_GROUP_BY
解决：如果使用Navicat，可F6进入命令行模式，输入 
					select @@GLOBAL.sql_mode;
查看当前的设置，然后输入 
set GLOBAL sql_mode='STRICT_TRANS_TABLES,NO_ZERO_IN_DATE,NO_ZERO_DATE,ERROR_FOR_DIVISION_BY_ZERO,NO_AUTO_CREATE_USER,NO_ENGINE_SUBSTITUTION'
进行设置即可。
解释说明：

SQLSTATE[42S22]: Column not found: 1054 Unknown column 'type' in 'where clause'
原因：在查询时where条件中写了一个不存在的字段
解释说明：

Incorrect integer value: '' for column 'id' at row 1
原因：5以上的版本如果是空值应该要写NULL
解决： 
修改my.ini中的sql-mode="STRICT_TRANS_TABLES,NO_AUTO_CREATE_USER,NO_ENGINE_SUBSTITUTION"，
			改为sql-mode="NO_AUTO_CREATE_USER,NO_ENGINE_SUBSTITUTION"
