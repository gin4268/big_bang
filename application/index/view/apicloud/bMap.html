bMap

移除覆盖物 地图事件 获取两点距离 根据经纬度查找中文地址 添加标注 添加标注2 移动标注 获取标注
移除覆盖物：
map.removeOverlay({
	ids: [1, 2, 3, 4, 5]
});
解释说明：

地图事件：
// 长按事件 
map.addEventListener({
	name: 'longPress'
}, function(ret) {
	if (ret.status) {
		alert(JSON.stringify(ret));
	}
});
// 地图拖动事件 
map.addEventListener({
	name: 'viewChange'
}, function(ret) {
	if (ret.status) {
		alert(ret.lon+ret.lat);
	}
});
// 添加点击事件 
map.setBubble({
	id: '100'+data[j].id,  // 覆盖物id
	bgImg: 'widget://image/Image 1.png',  // 背景图片
	content: {
		title: '大标题',
		subTitle: '概述内容',
		illus: 'widget://image/Image 1.png'  // 边上的图片
	},
	styles: {
		titleColor: '#000',
		titleSize: 16,
		subTitleColor: '#999',
		subTitleSize: 12,
		illusAlign: 'left'
	}
}, function(ret) {
  if (ret) {
	  alert(JSON.stringify(ret));
  }
});
解释说明：

获取两点距离：
map.getDistance({
	start: {
		lon: 106.486654,
		lat: 29.490295
	},
	end: {
		lon: 106.581515,
		lat: 29.615467
	}
}, function(ret) {
	if (ret.status) {
		alert(ret.distance);
	}
});
解释说明：
单位:米

根据经纬度查找中文地址：
map.getNameFromCoords({
	lon: 116.384767,
	lat: 39.989539
}, function(ret, err) {
	if (ret.status) {
		alert(JSON.stringify(ret));
	}
});
解释说明：

添加标注：
map.addAnnotations({
	annotations: [{
		id: 1,
		lon: 116.297,
		lat: 40.109
	}, {
		id: 2,
		lon: 116.29,
		lat: 40.109
	}],
	icon: 'widget://',
	draggable: true
}, function(ret) {
	if (ret) {
		alert(ret.id);
	}
});
解释说明：
draggable表示可拖动(通过长按),此标注不可直接移动,可添加点击事件

添加标注2：
map.addMobileAnnotations({
	annotations: [{
		id: 10,
		lon: 116.297,
		lat: 40.109,
		icon: 'widget://image/bMap_car1.png'
	}]
});
解释说明：
可旋转移动,不可直接添加点击事件

移动标注：
map.moveAnnotation({
    id: id,
    duration: 6,
    end: {
        lon: 116.3843839609304,
        lat: 39.98964439091298
    }
}, function(ret, err) {
    alert(ret.id + '移动结束');
});
解释说明：
只能移动MobileAnnotations类型的标注
